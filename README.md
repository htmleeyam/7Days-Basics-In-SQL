# 📊 SQL Basics in 7 Days
A beginner-friendly guide to learning SQL step by step in just one week. Each day covers key concepts with practical exercises!

## Day 1: Setup & First Database

- Install SQL on your PC (MySQL recommended, but SQLite is fine for learning)
- Create your first database and tables
- Insert sample data and run basic SELECT queries
- Practice basic commands: CREATE TABLE, INSERT, SELECT, WHERE
